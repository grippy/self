# self

A repository implementing a [DID:web](https://w3c-ccg.github.io/did-method-web/) (Decentralized Identifier) document.

## DID Identifier

```
did:web:raw.githubusercontent.com:grippy:self
```

## Quick Access

- **DID Document**: [.well-known/did.json](.well-known/did.json)
- **Documentation**: [DID.md](DID.md)
- **Raw URL**: https://raw.githubusercontent.com/grippy/self/main/.well-known/did.json

## What is DID:web?

DID:web is a decentralized identifier method that uses the web infrastructure (HTTPS, DNS) to provide a simple way to create and resolve DIDs. This repository hosts a DID document that can be resolved from GitHub's raw content delivery.

See [DID.md](DID.md) for detailed documentation on how to construct and use the DID:web identifier.
